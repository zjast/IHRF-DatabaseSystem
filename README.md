# 🐹 International Hamster Race Federation Database System
Projekt systemu bazy danych zaprojektowany dla Międzynarodowej Federacji Wyścigów Chomików (IHRF). System zarządza danymi dotyczącymi zawodników (chomików), pracowników, sponsorów, finansowania oraz wyników zawodów w ramach Światowej Olimpiady Chomików (WHO).

## O projekcie
Celem projektu było stworzenie bazy danych oraz analizy dla federacji sportowej zajmującej się dwiema kategoriami wyścigów:

* Kategoria Naturalna: Maraton, Sprint, Labirynt, Bieg z przeszkodami, Skok w dal, Bieg z obciążeniem oraz Limbo.

* Formuła Ch: Wyścigi w miniaturowych pojazdach napędzanych kołowrotkiem: Wyścig, Wyścig wytrzymałościowy oraz Drag Race.

## 🛠 Technologie
W projekcie wykorzystano następujące technologie:
- Język programowania Python z następującymi bibliotekami:
    * sqlalchemy
    * faker
    * urlib
    * pandas
    * mysql
    * plotly
- SQL z bazą mariaDB
- Język R z następującymi pakietami:
    * DBI
    * RMariaDB
    * plotly
    * dplyr

## 📊 Analiza Danych
W ramach projektu przeprowadzono analizę odpowiadającą na następujące pytania:
* Znajdź najlepszego chomiczego sportowca w każdej z kategorii, ile czasu trwa „szczyt kariery” u chomików?
* Czy któraś kategoria lub dyscyplina daje przewagę bogatym właścicielom chomików, którzy mogą sobie pozwolić na lepszy sprzęt?
* Sporządź wykres liczby uczestników zawodów, czy ten sport staje się bardziej, czy mniej popularny? Czy federacja jest rentowna finansowo? Które kategorie są najbardziej atrakcyjne dla widzów i sponsorów?       
* Czy stosowanie niedozwolonych substancji gwarantuje lepsze wyniki?
